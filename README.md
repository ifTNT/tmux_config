###Requirement
* tmux = 1.9
* vim
* tmux-mem-cpu-load
  * cmake
* powerline

###Installation
1. $ cd tmux-mem-cpu-load
2. $ cmake .
3. $ make
4. \# make install
5. $ cp ../battery /usr/local/bin/
6. copy the configuration of tmux and vim to your home
